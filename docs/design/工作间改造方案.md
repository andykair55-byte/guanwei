### 7.2 存在的问题

| # | 问题 | 严重度 | 说明 |
|---|------|--------|------|
| 1 | **碎片无持久化** | 🔴 高 | 当前碎片存在组件 `useState` 中，刷新页面即丢失。需要新建 `fragmentStore.ts`（Zustand + persist） |
| 2 | **引用到写作区未真正连接内容** | 🟡 中 | 点击碎片上的 `+` 按钮只在 textarea 末尾追加文本，没有做选区插入/引用块格式化 |
| 3 | **瓜田入口未改造** | 🟡 中 | `MelonDetailPage.tsx` 上的"写佐证"按钮还是旧文案，未改成"来分析一下" |
| 4 | **上下文自动带入未对接真实数据** | 🟡 中 | 当前 `?melonId=xxx` 只用了本地 mock 数据，未对接瓜田 store 的真实瓜内容 |
| 5 | **骨架生成仍基于主题而非碎片** | 🟢 低 | `creationService.ts` 的 skeleton prompt 仍是主题→输出，未利用已收集的碎片 |
| 6 | **编辑器仍是纯 textarea** | 🟢 低 | 无 Markdown 预览、无引用块样式、无内联核查标记 |
| 7 | **参考面板在移动端体验粗糙** | 🟢 低 | 滑动抽屉已实现但切换不够流畅，键盘弹出时布局抖动 |
