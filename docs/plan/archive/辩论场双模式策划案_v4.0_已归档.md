> 最后更新：2026-07-15
> 前置版本：v3.0《辩论体系策划案》（已归档，本版替代）

---

## 一、改造背景

### 1.1 问题诊断

| 问题 | 原因 |
|------|------|
| 辩论场页面入口混乱 | 之前只有1个入口，把"娱乐辩论"和"国赛辩论"混在一起 |
| "圆桌局"无意义 | 没有独立玩法价值，已删除 |
| "4v4 国赛"名不副实 | 实际是辩论大厅（房间列表/匹配），不是国赛机制 |
| 国赛沦为"菜市场吵架" | 缺乏强制顺序、发言锁定、严格计时，变成你一句我一句的混战 |

### 1.2 设计原则

1. **两个入口，两种模式**：娱乐辩论和国赛辩论是完全独立的体验，各有入口、各有大厅、各有房间
2. **模式定位清晰**：
   - 娱乐 = 游戏化社交，轻松、快节奏、人人参与
   - 国赛 = 严肃竞技，深度思考、严格流程、专业评判
3. **先搭架子后填玩法**：本次先完成入口/路由/大厅骨架，具体玩法机制待市场研究后细化

## 二、整体架构

### 2.1 入口结构

```
娱乐大厅 (/entertainment)
  └── 真人辩论厅 (分类)
        ├── 娱乐辩论 → /entertainment/debate          （模式1）
        └── 国赛辩论 → /entertainment/debate/national  （模式2）
```

### 2.2 两种模式对比

| 维度 | 模式1：娱乐辩论大厅 | 模式2：专业国赛辩论 |
|------|---------------------|---------------------|
| **定位** | 游戏化社交辩论，轻松娱乐 | 严肃竞技，深度思考 |
| **人数** | 1v5（6人），不对称对抗 | 4v4（8人），对称对抗 |
| **核心体验** | 人人发言、投票博弈、快节奏 | 严格流程、不可打断、评委打分 |
| **节奏** | 快，多轮，字数递增 | 慢，6环节固定流程 |
| **评判** | AI每轮评分 + 谬误检测 | 3评委四维度打分 |
| **趣味点** | 投票淘汰、入场费博弈 | 评委点评、MVP评选 |
| **社交感** | 像狼人杀房间，和朋友玩 | 像看辩论赛直播，有仪式感 |

## 三、模式1：娱乐辩论大厅（1v5）

### 3.1 基本设定

- **人数**：1v5，共6人
- **补位**：人不够时 AI 自动补位
- **形式**：借鉴狼人杀的房间社交感，但本质是辩论而非身份推理
- **每个玩家都有发言机会**

### 3.2 待研究问题（标记，不急于定论）

| 编号 | 问题 | 当前思路 |
|------|------|----------|
| Q1-1 | "1v5"中的1是固定持少数派观点，还是每轮动态变化？ | 倾向固定（1人持观点 vs 5人质疑） |
| Q1-2 | 发言顺序是1人陈述→5人轮流质疑→1人回应，还是自由混战？ | 待研究 |
| Q1-3 | "狼人杀元素"具体指什么——投票淘汰？角色分配？还是仅社交氛围？ | 待研究 |
| Q1-4 | 投票淘汰机制：踢谁？怎么投？被踢后能否观战？ | 待研究 |
| Q1-5 | 胜负判定：1人坚持到底算赢？还是5人说服1人算赢？ | 待研究 |
| Q1-6 | 入场费/积分机制是否保留？ | 待研究 |

## 四、模式2：专业国赛辩论（4v4）

### 4.1 基本设定

- **人数**：4v4，共8人
- **补位**：人不够时 AI 自动补位
- **赛制**：华语辩论世锦赛标准 4v4 赛制
- **核心目标**：解决"菜市场吵架"——强制顺序、不可打断、严格计时

### 4.2 六环节流程

```
① 开篇立论 → ② 攻辩环节 → ③ 攻辩小结 → ④ 自由辩论 → ⑤ 总结陈词 → ⑥ 评委评议
```

| 环节 | 时长 | 规则 | 防吵架设计 |
|------|------|------|------------|
| ① 开篇立论 | 各3分钟 | 正方一辩→反方一辩，依次完整陈述 | 发言锁定，其他人无法插入 |
| ② 攻辩环节 | 提问30s/回答60s | 反二→正一、正二→反一、反三→正二、正三→反二 | 仅此环节允许提问方打断 |
| ③ 攻辩小结 | 各1.5分钟 | 反方一辩→正方一辩 | 不可引入新论点 |
| ④ 自由辩论 | 各4分钟 | 双方交替，每人30秒 | 超时自动截止，一方时间耗尽后另一方单独发言 |
| ⑤ 总结陈词 | 各3.5分钟 | 反方四辩→正方四辩 | 不可引入新论点 |
| ⑥ 评委评议 | — | 3位评委独立打分 | 评分维度：逻辑30%/论据25%/反驳25%/表达20% |

### 4.3 防吵架核心机制

1. **强制顺序发言**：每个环节有固定发言顺序，不可抢话
2. **发言锁定**：轮到你时才解锁输入框，其他人只能观看
3. **不可打断**（攻辩环节除外）：发言方有完整时间，其他人无法插入
4. **思考缓冲**：AI发言前显示"思考中..."，人类发言前有准备倒计时
5. **严格计时**：超时自动截止，输入框锁定
6. **攻辩特殊规则**：只有攻辩环节允许提问方打断，其他环节一律完整发言

### 4.4 待研究问题

| 编号 | 问题 | 当前思路 |
|------|------|----------|
| Q2-1 | 评委是纯AI还是有真人评委模式？ | MVP用AI模拟3位风格不同的评委 |
| Q2-2 | 是否需要录像/回放功能？ | MVP不做，用发言记录替代 |
| Q2-3 | 辩手是否能看到对方发言？ | 是，实时展示在舞台区 |
| Q2-4 | 观众能否发弹幕？ | 待研究，可能影响严肃氛围 |
| Q2-5 | 是否需要赛前准备阶段（看资料）？ | 待研究 |
| Q2-6 | Demo模式加速倍率是否合理？ | 当前0.15倍，需实测调整 |
