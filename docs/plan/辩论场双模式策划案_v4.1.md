## 一、改造背景

### 1.1 问题诊断

| 问题 | 原因 |
|------|------|
| 辩论场页面入口混乱 | 之前只有1个入口，把"娱乐辩论"和"国赛辩论"混在一起 |
| "圆桌局"无意义 | 没有独立玩法价值，已删除 |
| "4v4 国赛"名不副实 | 实际是辩论大厅（房间列表/匹配），不是国赛机制 |
| 国赛沦为"菜市场吵架" | 缺乏强制顺序、发言锁定、严格计时，变成你一句我一句的混战 |

### 1.2 设计原则

1. **两个入口，两种模式**：娱乐辩论和国赛辩论是完全独立的体验，各有入口、各有大厅、各有房间
2. **模式定位清晰**：
   - 娱乐 = 游戏化社交，轻松、快节奏、人人参与
   - 国赛 = 严肃竞技，深度思考、严格流程、专业评判
3. **先搭架子后填玩法**：本次先完成入口/路由/大厅骨架，具体玩法机制待市场研究后细化

## 二、整体架构

### 2.1 入口结构

```
娱乐大厅 (/entertainment)
  └── 真人辩论厅 (分类)
        ├── 娱乐辩论 → /entertainment/debate          （模式1）
        └── 国赛辩论 → /entertainment/debate/national  （模式2）
```

### 2.2 两种模式对比

| 维度 | 模式1：娱乐辩论大厅 | 模式2：专业国赛辩论 |
|------|---------------------|---------------------|
| **定位** | 游戏化社交辩论，轻松娱乐 | 严肃竞技，深度思考 |
| **人数** | 1v5（6人），不对称对抗 | 4v4（8人），对称对抗 |
| **核心体验** | 人人发言、投票博弈、快节奏 | 严格流程、不可打断、评委打分 |
| **节奏** | 快，多轮，字数递增 | 慢，6环节固定流程 |
| **评判** | AI每轮评分 + 谬误检测 | 3评委四维度打分 |
| **趣味点** | 投票淘汰、入场费博弈 | 评委点评、MVP评选 |
| **社交感** | 像狼人杀房间，和朋友玩 | 像看辩论赛直播，有仪式感 |

## 三、模式1：娱乐辩论大厅（6人独立麦位，非1v5）

> 2026-07-15 修正：不是"1人 vs 5人"的少数派/多数派对抗结构，而是 **6个完全独立的个体**，各自可能持不同立场，没有固定阵营。原"1v5"框架隐含的"谁说服谁"胜负逻辑不再成立，已改为个人认可度积分制（见3.4）。

### 3.1 基本设定

- **人数**：6个麦位，每人独立发言、独立观点，不预设阵营
- **补位**：麦位空出后，从候补观众队列中自动补位（技术上复用语聊房通用的麦位状态机：观众 → 排队中 → 麦上 → 下麦/被踢 → 观众）
- **形式**：借鉴狼人杀/语聊房的房间社交感，本质是"人人都能被听见"的表达场，而非身份推理或阵营对抗
- **核心驱动力**：用户的"表达欲/被看见欲"，不是"赢辩论"

### 3.2 核心循环（本轮定案）

```
观众打字提交观点
   ↓
AI审核：判断"善意认真表达" vs "骂人/无实质内容/纯发泄"
   ├─ 未通过 → 留在候补池，可修改后重提
   └─ 通过 → 进入排队（FIFO，先到先得，不设付费插队）
   ↓
麦位空出 → 按队列顺序自动补位
   ↓
麦上发言：基础时长保底（初定45秒，具体数值需小流量实测调整）
   ↓
时长到 → AI给该段发言打"认真度/信息量"分（不是"有趣程度"评分）
   ├─ 高分 → 获得续麦权（免排队再讲一轮，次数封顶，防止长期霸麦）
   └─ 普通分 → 回观众池重新排队
   ↓
观众端权限：仅"送花/点赞"等正向助推（影响下次排队优先级）
           不设"投票下麦"权限
```

**关键设计决策：观众端不设"投票下麦"权限**
目标用户是高中生/大学生，产品红线又排除政治/性别对立等高风险话题——一旦给观众"踢人"权力，最容易演变的不是"淘汰观点差的人"，而是对表达方式/外形不讨喜的人的群体攻击，这与"防止娱乐辩论沦为菜市场吵架"是同一类风险换了个受害对象。因此把负向清退完全交给"计时器 + AI"这类中性规则，观众端只保留正向权限。

### 3.3 违规处理（骂人 / 绕过AI审核）

- **AI实时监测 + 观众举报双通道并行**：AI预筛无法100%拦住"先规避措辞再骂人"的用户，实时监测持续监听麦上发言，补上"过审后才开始违规"的漏洞；观众举报作为AI漏检的兜底。两条通道都不需要人工审核团队，AI复审即可处理。
- **处罚阶梯**：首次违规 → 立即掐麦 + 警告；短时间内多次触发 → 进入冷却期，暂时无法再申请上麦。不建议首次即封号，娱乐社交场景的容错尺度应比国赛更宽松。

### 3.4 认可度积分（替代原"胜负判定"）

不做二元输赢判定。每次发言获得的点赞/AI认可分累积为个人分数，用于：
- 影响下次排队优先级（分高的人更快轮到）
- 生成本场"高光发言"榜单

这个设计直接对应"变现欲"这一核心洞察：用户要的是被更多人看见、被认可，不是打赢一场辩论。积分和榜单把这个心理诉求显性化、可视化。

## 四、模式2：专业国赛辩论（4v4）

### 4.1 基本设定

- **人数**：4v4，共8人
- **补位**：人不够时 AI 自动补位
- **赛制**：华语辩论世锦赛标准 4v4 赛制
- **核心目标**：解决"菜市场吵架"——强制顺序、不可打断、严格计时

### 4.2 六环节流程

```
① 开篇立论 → ② 攻辩环节 → ③ 攻辩小结 → ④ 自由辩论 → ⑤ 总结陈词 → ⑥ 评委评议
```

| 环节 | 时长 | 规则 | 防吵架设计 |
|------|------|------|------------|
| ① 开篇立论 | 各3分钟 | 正方一辩→反方一辩，依次完整陈述 | 发言锁定，其他人无法插入 |
| ② 攻辩环节 | 提问30s/回答60s | 反二→正一、正二→反一、反三→正二、正三→反二 | 仅此环节允许提问方打断 |
| ③ 攻辩小结 | 各1.5分钟 | 反方一辩→正方一辩 | 不可引入新论点 |
| ④ 自由辩论 | 各4分钟 | 双方交替，每人30秒 | 超时自动截止，一方时间耗尽后另一方单独发言 |
| ⑤ 总结陈词 | 各3.5分钟 | 反方四辩→正方四辩 | 不可引入新论点 |
| ⑥ 评委评议 | — | 3位评委独立打分 | 评分维度：逻辑30%/论据25%/反驳25%/表达20% |

### 4.3 防吵架核心机制

1. **强制顺序发言**：每个环节有固定发言顺序，不可抢话
2. **发言锁定**：轮到你时才解锁输入框，其他人只能观看
3. **不可打断**（攻辩环节除外）：发言方有完整时间，其他人无法插入
4. **思考缓冲**：AI发言前显示"思考中..."，人类发言前有准备倒计时
5. **严格计时**：超时自动截止，输入框锁定
6. **攻辩特殊规则**：只有攻辩环节允许提问方打断，其他环节一律完整发言
