# 见微 —— 产品设计文档

> **Slogan：** 见微知著，明辨是非
> **使命：** 用有趣的方式提升全民信息素养

---

## 一、项目背景与社会意义

### 1.1 问题现状

我们生活在一个信息爆炸的时代，但大多数人缺乏足够的信息辨识能力：

- **老年人**：数字鸿沟导致容易被诈骗、被虚假养生信息误导
- **青少年**：心智尚未成熟，容易被网络谣言、营销号带节奏
- **中青年人**：跨领域信息差导致容易踩坑，术业有专攻但骗子专攻人心
- **全社会**：情绪先行、事实靠后，男女对立、网络暴力愈演愈烈

国家反诈APP推广多年，为什么依然有人被骗？核心原因是：**没有人愿意"主动学习反诈"。** 人们工作生活已经很累了，为什么还要打开一个APP"上课"？

### 1.2 我们的答案

**让提升判断力这件事，变得像"吃瓜"一样有趣。**

不是说教，不是培训，而是用游戏化的方式，让用户在娱乐中潜移默化地养成"先求证、后相信"的思维习惯。

### 1.3 回应时代焦虑

AI时代，人们普遍存在四大焦虑，也是本次大赛的核心关切：

| 焦虑 | 我们的回答 |
|---|---|
| **AI太强了，人还需要学习吗？** | 需要。AI给的是答案，人需要的是判断力。见微训练的不是知识，是"辨别知识真伪"的元能力 |
| **AI信息会被污染，怎么信？** | 我们不生产"答案"，我们呈现"证据链"。每个结论都标注信息来源，让用户自己判断可信度 |
| **人类会过度依赖AI吗？** | 我们坚持"AI辅助，人做决策"。AI只提供工具和证据，最终判断权永远在人手里 |
| **AI会取代人的工作吗？** | 不会。AI是放大器，不是替代者。见微的内容审核机制就是例子——AI做初筛提效率，人工做终审保质量，反而创造了新的岗位形态 |

AI降低了编程门槛，本质是让AI能更快融入社会。我们要找的，是人与AI和谐共处的"两全法"。

## 二、产品定位与核心价值

### 2.1 产品定位

一个"理性吃瓜"的信息求证社区。

表面上是娱乐社区（猜真假、赚积分、升段位），内核是AI驱动的信息素养训练平台。

### 2.2 核心价值主张

1. **有趣** —— 吃瓜是人性刚需，游戏化降低使用门槛
2. **有用** —— AI信息求证工具箱，真能解决实际问题
3. **有温度** —— 公益板块，让流量变成温暖的力量
4. **有成长** —— 不知不觉中提升思辨能力和信息素养

### 2.3 目标用户

**全年龄段、全职业覆盖：**

- **青少年**：爱玩、好奇心强，在游戏中培养思辨习惯
- **中青年**：吃瓜放松，同时用工具箱辅助工作和决策
- **老年人**：大字版、语音朗读，科普类内容帮助防骗
- **机关单位/企业**：可用作员工信息安全培训、反诈教育

## 四、核心玩法：瓜田与真假擂台

### 4.1 瓜田信息流

- 瀑布流展示各类"待求证"内容
- 算法推荐，越刷越懂用户兴趣
- 分类筛选：娱乐 / 科技 / 生活科普 / 社会热点 / 历史 / 财经 / 更多
- 每个卡片显示：标题、封面、当前押注比例、剩余开奖时间

### 4.2 真假押注玩法

**核心机制：**
1. 用户看到一个"瓜"，选择押"真"或"假"
2. 下注积分（可选10/50/100三档，或自由选择）
3. AI多Agent系统去全网搜集证据、交叉验证
4. 开奖时间到，公布AI生成的"实锤报告"
5. **猜对的用户**：瓜分奖池积分，获得对应奖励
6. **猜错的用户**：不扣积分，但需要观看1分钟科普讲解视频作为"惩罚"

**设计理念：**
- 不扣积分 = 降低心理负担，用户敢于尝试
- 看科普 = 把"学习"变成游戏机制的一部分，寓教于乐
- 押注有时间限制 = 制造紧迫感，促使用户回来

### 4.3 每日精选

- 每天精选3-5个高质量"瓜"
- 参与每日精选可获得额外积分奖励
- 培养用户"每天来看看"的使用习惯

## 五、AI求证工具箱

### 5.1 核心能力

| 功能 | 说明 | 额度 |
|---|---|---|
| **一键求证** | 输入文字/链接，AI快速核查信息真伪 | 每日免费3次 |
| **多模态识别** | 图片真伪鉴定、截图识别、视频溯源 | 积分兑换/会员 |
| **深度调查** | 全面的资料搜集与分析，生成详细报告 | 积分兑换/会员 |
| **报告导出** | 将求证结果导出为结构化文档 | 积分兑换/会员 |

### 5.2 实锤报告结构

每份AI生成的求证报告包含：

1. **信息时间线** —— 事件发展脉络，信息最早出处
2. **证据链列表** —— 逐条列出搜集到的证据
3. **来源可信度标注** —— 每个信息源标注权威等级
   - ⭐⭐⭐⭐⭐ 政府网站、权威媒体、学术论文
   - ⭐⭐⭐⭐ 正规媒体、知名机构
   - ⭐⭐⭐ 自媒体、论坛
   - ⭐⭐ 未知来源、匿名信息
4. **关键疑点分析** —— 信息中的矛盾点、不合理之处
5. **倾向性判断** —— "根据现有公开信息，倾向于认为是XX"（留有余地，不说死）
6. **免责声明** —— AI核查仅供参考，不构成任何建议

### 5.3 技术原则：AI辅助，而非AI决策

1. **只给证据，不给绝对结论** —— 呈现事实，判断在人
2. **敢于说"不知道"** —— 信息不足时明确说明，不编造
3. **区分"事实"与"推测"** —— 哪些是确认的，哪些是分析的，标注清楚
4. **可追溯、可验证** —— 每个信息点都标注来源，用户可以自行核实

## 六、社区生态

### 6.1 内容类型

| 类型 | 说明 |
|---|---|
| **求锤帖** | "我看到一个瓜，大家帮我鉴别是真是假" |
| **询问帖** | "我有个问题，有人知道答案吗" |
| **分享帖** | "我查到了一些信息，分享给大家" |
| **讨论帖** | 针对某个话题的开放式讨论 |

### 6.2 积分闭环

```
用户发帖 → 可用积分购买曝光 → 更多人看到 →
更多人参与 → 高质量回答/实锤获得积分奖励 →
积分可用于：买曝光 / 换工具箱功能 / 公益捐赠 →
形成内容生产与消费的正向循环
```

### 6.3 内容审核机制

**AI初筛 + 人工终审**

- **AI审核员**：快速识别敏感内容（政治、暴力、色情、隐私、煽动对立等）
- **人工审核员**：AI不确定的内容，以及高热度内容的复核
- **用户举报**：社区用户共同监督，举报核实有积分奖励

**设计理念：**
- AI不是来取代审核员的，是来提升效率的
- AI做初筛，人做判断，人机协作，各展所长
- 反而创造了新的就业形态和岗位需求

### 6.4 社区调性

与知乎等平台的差异化：

| 平台 | 权威来源 | 氛围 |
|---|---|---|
| 知乎 | 人（学历、身份、头衔） | 精英感、门槛高 |
| 见微 | 证据（来源、逻辑、数据） | 平等、重事实、轻身份 |

在见微，不看你是什么学历什么身份，只看你说的有没有证据、逻辑通不通。

## 七、成长体系

### 7.1 积分系统

**获取积分：**
- 每日登录 +10
- 猜对瓜 +20~100（根据难度和下注）
- 发布高质量内容 +奖励
- 回答被评为优质 +奖励
- 邀请好友 +50
- 举报核实 +奖励

**使用积分：**
- 购买帖子曝光度
- 兑换工具箱高级功能
- 解锁头像框、称号等装饰
- 公益捐赠（积分折算为真实公益基金）

**不扣积分：**
- 猜错瓜不扣分，只看科普小视频

### 7.2 段位系统

从低到高：
- 吃瓜群众
- 瓜田新手
- 鉴瓜学徒
- 瓜田侦探
- 鉴瓜达人
- 鉴瓜大师
- 见微先知

升级依据：
- 鉴瓜准确率
- 参与次数
- 贡献内容质量
- 社区活跃度

### 7.3 成就徽章

- 火眼金睛（连续猜对10次）
- 谣言粉碎机（累计正确识别50个假消息）
- 公益达人（累计捐赠XX积分）
- 社区卫士（举报XX次有效）
- 等等...

## 八、公益板块：暖心见微

### 8.1 板块定位

让流量变成温暖的力量。用社区的注意力资源，帮助真正需要帮助的人。

### 8.2 核心内容

- **寻人启事**：AI核实信息真实性后发布，用户可转发扩散
- **求助信息**：经过验证的困难求助，用户可捐赠积分
- **公益项目**：联合公益机构，上线正规公益项目
- **好人好事**：传播正能量，记录社会温暖瞬间

### 8.3 参与方式

- **转发扩散**：用户转发公益内容，扩大传播
- **积分捐赠**：用户积分可折算为真实资金捐赠（平台配套资金）
- **志愿服务**：后续可拓展线上线下志愿服务对接

### 8.4 为什么做公益？

1. **社会价值**：让产品不止于娱乐，更有温度和使命感
2. **用户认同**：用户会因为"这个产品在做好事"而产生好感和忠诚度
3. **覆盖全龄**：老年群体尤其关注公益、寻人等内容
4. **品牌提升**：提升产品格调，获得社会认可

## 九、技术架构亮点

### 9.1 多Agent协作系统

不是单一Agent，而是多个Agent分工协作：

| Agent角色 | 职责 |
|---|---|
| **搜集Agent** | 自动全网检索，抓取相关信息 |
| **验证Agent** | 交叉验证信息来源，评估可信度 |
| **分析Agent** | 整理证据链，生成结构化报告 |
| **审核Agent** | 初筛敏感内容，降低人工成本 |
| **多模态Agent** | 处理图片、视频、音频等非文本内容 |

### 9.2 信息可信度评估算法

- **来源分级**：政府网站 > 权威媒体 > 学术论文 > 正规媒体 > 自媒体 > 未知来源
- **交叉验证**：多个独立来源相互印证的信息，可信度更高
- **传播溯源**：追踪信息最早出处，分析传播路径
- **情感分析**：识别是否存在煽动情绪、带节奏的话术
- **事实核查库**：持续积累已验证的真假信息库，提升核查效率

### 9.3 多模态识别能力

- 图片真伪鉴定（P图检测、旧图新用识别）
- 视频溯源（是否AI生成、是否拼接剪辑）
- 截图内容识别与核查
- 文本AI生成检测

## 十一、竞品差异化

| 维度 | 见微 | ChatGPT/豆包等 | 抖音求真 | 国家反诈APP |
|---|---|---|---|---|
| 定位 | 游戏化信息求证社区 | 通用对话助手 | 平台功能 | 官方工具 |

| 使用门槛 | 极低、好玩、停不下来 | 中等、需要会提问 | 低、但被动 | 高、说教感强 |
| 社会价值 | 提升全民信息素养 | 提升效率 | 辟谣 | 反诈宣传 |
| 商业潜力 | 社区+工具+公益，想象空间大 | SaaS、API | 广告 | 财政支持 |

## 十三、总结

见微不是又一个AI聊天工具，也不是又一个反诈APP。

它是一次实验：**能不能用人们喜欢的方式，去做一件对社会有意义的事？**

- 用"吃瓜"的外壳，装下"求真"的内核
- 用游戏化的机制，实现教育的目的
- 用AI的能力，放大人的判断力
- 用流量的力量，传递温暖的价值

见微知著，明辨是非。

我们相信，当每个人都学会"先看看证据再下结论"的时候，这个世界会好一点。
